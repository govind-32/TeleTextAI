# TeletextAI - Product Overview

TeletextAI is a modern web application that reimagines the classic 1980s teletext experience with AI-powered features. It presents information through a retro CRT-style interface with authentic teletext aesthetics.

## Core Concept
- Nostalgic teletext UI with page-based navigation (3-digit page numbers like P100, P101)
- CRT visual effects including scanlines, glow, and glitch transitions
- AI-enhanced content: headlines summarization, pixel art generation, chat assistant

## Key Pages
- **P100** - Home: Navigation hub with page tiles
- **P101** - AI Headlines: AI-summarized news
- **P102** - Weather: City weather lookup (OpenWeatherMap API)
- **P103** - Crypto: Cryptocurrency prices
- **P105** - Pixel Art: AI-generated 8x8 pixel art
- **P199** - Assistant: AI chat interface
- **P900** - Settings: CRT intensity, volume, theme

## User Interaction
- Keyboard-driven: number keys for page entry, arrow keys for prev/next
- Tab navigation for focusable elements within pages
- Audio feedback on page transitions
- Persistent user settings via localStorage
